# Velmora iPhone Setup

Набор базовых настроек и идей для iPhone:
- минимум уведомлений;
- закрытый круг общения;
- аккуратное использование соцсетей;
- спокойный рабочий режим.

## Что внутри

- описания нужных настроек iOS;
- идеи для режимов «Фокус»;
- черновики автоматизаций через «Команды» (Shortcuts).

## Зачем

Чтобы телефон не мешал, не светил лишнюю информацию и был настроен под личный, закрытый режим использования.
